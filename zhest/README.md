# Жесткие системы ОДУ, вариант 4.2

Решается уравнение Релея

`x'' - mu * (1 - (x')^2) * x' + x = 0`,

где `mu = 1000`, `x(0) = 0`, `x'(0) = 0.001`, `T = 1000`.

Оставлено только то, что требуется в задании:

- методы `Radau IIA` порядков `1`, `3`, `5`;
- метод `SDIRK2`;
- графики `x(t)`, `v(t)` и фазовый портрет;
- сравнение с библиотечным решением `SciPy Radau`;
- график ошибок по шагу;
- области устойчивости всех используемых методов.

## Файлы

- `rayleigh_solver.py` — основной модуль;
- `zhectkie.py` — запуск расчета;
- `rayleigh_assignment.ipynb` — короткий ноутбук-демонстрация;
- `results_variant_4_2/` — итоговые изображения и отчёт.

## Запуск

```bash
MPLCONFIGDIR=.mplconfig ./.venv/bin/python zhectkie.py
```
